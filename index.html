Welcome.

Este es mi portfolio personal, tiene como objetivo mostrar mis trabajos y habilidades, como así también mi progreso en el dinámico mundo del desarrollo web. Se trata de una singlepage desarrollada con HTMl y CSS. Tiene componentes de Bootstrap e íconos de Fontawesome, para lograr una experiencia amena para el navegante. Están anclados aquí los links de varios proyectos desarrollados, disponibles para ser visitados.
Al final de esta página te vas a encontrar con un formulario de contacto, para que puedas comunicarte en caso de que necesites de mis servicios.
